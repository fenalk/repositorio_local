# Git e GitHub mais Seguro
- Autenticação de Dois Fatores (2FA)
- Senhas com >8 caracteres, letras maiúsculas,minúsculas e símbolos
- Evitar nomes e palavras comuns
- Evitar Padrões
- Nunca compartilhe uma senha

# Git Branches ou Ramificações
Branch: assemelha-se a uma árvore. Branch significa o ramo principal que no linguajar do Git é conhecido como Master. Master é o ramo de produção, versão final.
